# ⏳ Pomodoro Focus-Timer Bot

Telegram-бот для фокус-сессий по методу Помодоро с красивой анимацией песочных часов и звуком.

## Возможности

| Фича | Описание |
|------|----------|
| 🎛 4 режима | Классический · Длинный фокус · Быстрый спринт · Глубокая работа |
| ⏳ Визуал | PIL-изображение песочных часов, меняется с каждой фазой |
| 🔊 Звук | Шум песка при старте · Колокольчик при завершении · Восходящий аккорд после перерыва |
| 📊 Статистика | Счётчик сессий · Общее время фокуса · Прогресс цикла |
| ⏭ Управление | Стоп · Пропустить фазу · Инлайн-кнопки |

## Режимы

| Режим | Фокус | Перерыв | Длинный перерыв | Цикл |
|-------|-------|---------|-----------------|------|
| 🍅 Классическое | 25 мин | 5 мин | 15 мин | 4 сессии |
| 🎯 Длинный фокус | 50 мин | 10 мин | 30 мин | 3 сессии |
| ⚡ Быстрый спринт | 15 мин | 3 мин | 10 мин | 4 сессии |
| 🧘 Глубокая работа | 90 мин | 20 мин | 40 мин | 2 сессии |

## Установка

```bash
# 1. Клонировать репо
git clone <repo-url>
cd pomodoro_bot

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Создать .env файл
cp .env.example .env
# Вставить токен от @BotFather в .env

# 4. Запустить
python bot.py
```

## Команды бота

| Команда | Действие |
|---------|----------|
| `/start` | Главное меню |
| `/mode` | Выбор режима |
| `/stop` | Остановить таймер |
| `/stats` | Статистика сессий |

## Дизайн

Каждая фаза имеет свою цветовую схему:
- 🍅 **Фокус** — тёмно-синий фон, золотисто-оранжевый песок, звёзды
- ☕ **Перерыв** — тёмно-зелёный, зелёный песок
- 🌙 **Длинный перерыв** — тёмно-фиолетовый, фиолетовый песок, звёзды

Песочные часы рисуются динамически:
- Верхняя половина опустошается по мере истечения времени
- Нижняя половина наполняется соответственно
- Тонкая струйка песка в горлышке
- Блики и искры на поверхности песка
- Свечение вокруг стекла

## Звуки

Все звуки генерируются программно при первом запуске и кешируются:
- **Шум песка** (старт фокуса) — отфильтрованный белый шум ~4 сек
- **Колокол** (конец фокуса) — затухающие синусоиды C5+E5
- **Восходящий аккорд** (конец перерыва) — C4→E4→G4 гамма
- **Тик** (для внутренних событий)
